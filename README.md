# CompaniepNet

API using ASP.NET CORE 6.

## Prerequisites

- .NET 6
- Visual Studio o VS Code

## Setup

#### Clone repo

`https://github.com/dluismendezpy/CompaniepNet.git`


#### Run project
- Using dotnet CLI

    `dotnet run`

- Using visual studio

    `Start Debugging` from the menu bar.
